아키텍쳐
cpu : 컴퓨터의 핵심 중앙 처리 장치
cpu = ALU(산술 논리 장치)+CU(제어 장치)+레지스터
1.레지스터 : 데이터를 저장할 수 있는 부분, RAM과 약간 비슷하지만 RAM만큼 많은 데이터를 보유할 수 없지만 훨씬 빠르게 작동한다.
a.명령어 레지스터 : 처리할 명령어의 주소를 ram에 저장.
b.메모리 주소 레지스터 : 처리할 데이터의 주소를 ram에 저장.
c.메모리 데이터 레지스터 : 처리 중인 데이터를 저장.
d.프로그램 카운터  : 다음 명령어의 ram에 주소를 저장.
e.누산기 : 계산의 결과를 저장.
2.CU : 각 명령어의 의미 해통 후 다른 구성 요소의 작동 방식을 제어.
ALU와 메모리가 수행해야 하는 작업을 신호로 알림.
3.ALU : 논리 연산을 수행하는 부분.

main memory, input-output 부분과 연결되는 부분은 CU, ALU이다.
레지스터와 ALU는 프로세서를 구성하고 서로 통신하고 CON
